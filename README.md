# Secure System Configuration and Device Hardening Project
Network Security Design – 3-Zone Segmentation &amp; Firewall Management Developed during the First Level Professional Master in Cybersecurity (2024/25), University of Pisa.
Designed and simulated a secure network infrastructure in Cisco Packet Tracer, implementing a three-zone security architecture (e.g., Inside, Outside, DMZ). Configured and managed firewall policies to ensure granular traffic control and robust network isolation.
